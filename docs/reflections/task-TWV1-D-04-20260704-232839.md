- Task: TWV1-D-04 Wire settings changes to live host repaint
- Encountered Problem: New colors would be useless if they only persisted to disk and did not refresh the taskbar widget immediately.
- Thought Process: I reused the existing settings-apply notification path instead of inventing a new widget-specific IPC contract.
- Options Considered: Add a new command just for widget visuals; poll config changes; extend the existing applied-keys and invalidate path.
- Chosen Solution: Extend the current settings bridge and applied-key list with `widget_visual.palette.*`.
- Rationale: The existing settings pipeline already knew how to refresh the host safely, so reusing it kept the change small and coherent.
