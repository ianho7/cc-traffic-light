- Task: GHI-B-03 implement JSON parsing and structure validation
- Encountered Problem: 第一版把 `ConvertFrom-Json -Depth` 用到了当前 PowerShell 版本不支持的参数上，表面看像 JSON 解析失败。
- Thought Process: 先确认 fixture 内容本身合法，再回查脚本对 PowerShell 版本的假设。
- Options Considered: 继续怀疑 fixture；保留 `-Depth`；改成兼容当前环境的基础 `ConvertFrom-Json` 并补结构校验。
- Chosen Solution: 去掉 `ConvertFrom-Json -Depth`，保留显式的顶层 `hooks` 结构校验。
- Rationale: 这样错误能真正指向输入结构，而不是被版本兼容性伪装成 parse failure。
