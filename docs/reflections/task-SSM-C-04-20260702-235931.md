- Task: SSM-C-04 为状态枚举和来源方法建立用户可见 label 映射
- Encountered Problem: 当前运行态里大量字符串是内部 key，例如 `tray_only`、`hook_state`、`untrusted`，不适合直接展示给用户。
- Thought Process: 把状态、方法、可信度、来源和 widget mount 都纳入同一套 label 映射，而不是在每个界面上手动 humanize。
- Options Considered: 继续局部 humanize 字符串；在 i18n 层建立显式映射。
- Chosen Solution: 在 `i18n.rs` 中增加 source/state/method/confidence/widget_mount 的统一 label 输出。
- Rationale: 显式映射比字符串拼写转换更稳，也更适合中英双语扩展。
