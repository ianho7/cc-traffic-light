- Task: SSM-F-01 修改 Open Settings 流程，默认打开 Slint settings
- Encountered Problem: 只做基础设施不改入口，用户仍然会继续看到旧 Win32 自绘窗口，迁移目标不成立。
- Thought Process: 在保持 fallback 的前提下，把 tray `Open Settings` 改成优先尝试 Slint runtime。
- Options Considered: 继续保持 Win32 为默认；直接完全切换且无 fallback；优先 Slint、失败回退 Win32。
- Chosen Solution: 采用“优先 Slint，失败回退 Win32”的入口切换。
- Rationale: 这既向目标前进，又避免一次性切断稳定 fallback。
