- Task: GHI-B-05 make dry-run output redacted and stable
- Encountered Problem: 真实用户 dry-run 需要证据，但不应把本地路径和无关细节直接写进文档。
- Thought Process: 输出里真正需要的是行为摘要、事件动作和是否存在配置，而不是绝对路径本身。
- Options Considered: 默认输出所有路径；完全不输出路径相关字段；默认脱敏并提供显式 `-ShowPaths` 调试开关。
- Chosen Solution: 默认把路径输出为 `<redacted>`，仅在本地 fixture 调试时使用 `-ShowPaths`。
- Rationale: 这样能兼顾安全摘要和本地排障可用性。
