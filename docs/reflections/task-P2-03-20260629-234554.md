# Task Reflection

- Task: `P2-03` 校验样式切换后是否仍残留不必要的顶层窗口语义
- Encountered Problem: 仅仅记录 `style_after` 十六进制值不够直观，无法快速判断是否还残留 caption/popup 等语义。
- Thought Process: 不做复杂样式解析器，只提取与“仍像顶层窗口”最相关的一组 bit。
- Options Considered: 只保留原始 style；新增残留顶层 bit 检查。
- Chosen Solution: 增加 `residual_top_level_style_bits` 和 `top_level_style_cleared`。
- Rationale: 这让后续排查可以更快区分“样式没清干净”和“宿主/定位问题”。
