- Task: P2-03 明确当前机器的“候选父窗口”和“位置参考窗口”，不要同时保留多套策略
- Encountered Problem: Win11 下可选参考点不止一个，若同时保留多套策略会马上失控。
- Thought Process: 把“谁来当父窗口”和“谁来当位置锚点”拆开决策，而不是用一个模糊的“任务栏相关窗口”概念糊过去。
- Options Considered: 用 `ReBarWindow32` 当父窗口；用 `Shell_TrayWnd` 当父窗口；同时保留双路径。
- Chosen Solution: 选 `Shell_TrayWnd` 作为候选父窗口，选 `TrayNotifyWnd` 作为位置锚点。
- Rationale: 这条路径在当前机器上最稳定，也最容易形成保守定位公式。
