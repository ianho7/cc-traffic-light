- Task: GHI-C-05 keep install scope away from notify and config.toml
- Encountered Problem: 仓库里已经有 notify 配置脚本和 inline TOML 示例，安装器容易顺手扩散到非本轮范围。
- Thought Process: P1 的核心只是用户级 `hooks.json` 安装，不需要同时迁移两套配置表示。
- Options Considered: 同时支持 `notify`；同时支持 `config.toml`；只处理用户级 `hooks.json`。
- Chosen Solution: 脚本只操作 `hooks.json`，完全不触碰 `notify` 和 inline TOML。
- Rationale: 这样 merge 规则和回滚语义更简单，也更符合当前 MVP 范围。
