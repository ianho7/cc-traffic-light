- Task: SSM-B-03 新建 settings_slint.rs 作为最小 Slint host
- Encountered Problem: 仅有 `.slint` 文件还不够，必须有 Rust 侧宿主承接创建、显示、隐藏和状态推送。
- Thought Process: 先建立一个不接主入口的过渡 host，暴露 create/show/hide/update 接口，让后续主流程切换更平滑。
- Options Considered: 等主入口切换时再写 host；先创建独立 host 模块。
- Chosen Solution: 新建 `settings_slint.rs`，提供最小生命周期和 snapshot/config 推送能力。
- Rationale: 把宿主层先搭起来，可以在不破坏现有 Win32 settings 的情况下单独验证 Slint integration。
