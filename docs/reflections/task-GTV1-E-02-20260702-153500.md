- Task: GTV1-E-02 实现当前用户级 autostart
- Encountered Problem: 现有 General 页中的 `Enable autostart` 只会改本地配置，没有真正把软件接进 Windows 当前用户启动链路，因此设置是“伪开关”。
- Thought Process: 根据已确认边界，自启动必须是当前用户级、无提权、单进程、纯 Win32。对比了计划任务、启动文件夹和 `Run` 注册表项之后，优先选择 `HKCU\\...\\Run`，因为它最直接、最符合 V1 复杂度。
- Options Considered: 1. 继续只保存布尔配置。2. 使用启动目录快捷方式。3. 使用 `HKCU\\Software\\Microsoft\\Windows\\CurrentVersion\\Run` 注册表项。
- Chosen Solution: 新增 `autostart.rs`，由 settings 开关直接写入或删除 `HKCU\\Software\\Microsoft\\Windows\\CurrentVersion\\Run\\CcTrafficLight`，并在启动时把注册表真实状态回填到配置对象。
- Rationale: `Run` 项不需要管理员权限，能满足“当前用户级自启动”的产品约束；同时把真实系统状态回填到 UI，避免配置文件和系统状态漂移后仍显示错误勾选。
