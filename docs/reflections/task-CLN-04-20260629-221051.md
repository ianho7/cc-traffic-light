- Task: CLN-04 确认错误信息可读，失败时能定位到句柄探测、嵌入或定位哪一层
- Encountered Problem: Win32 失败如果只返回“false”，几乎没有排错价值。
- Thought Process: 按阶段输出，让每类失败都能落到某一层。
- Options Considered: 继续使用模糊错误；写复杂错误系统；用阶段化日志表达失败层次。
- Chosen Solution: 通过 `taskbar probe failed`、`taskbar attach result`、`taskbar layout result` 等日志区分阶段。
- Rationale: 这已经足以让下一次问题定位不靠猜。
