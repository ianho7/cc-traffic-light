- Task: SSM-E-03 把语言切换事件接到持久化配置，并保证重启后仍保持上次语言
- Encountered Problem: 只有静态显示语言模式还不够，必须能从 Slint 直接切换并持久化。
- Thought Process: 先用 cycle 行为完成最小交互闭环，并直接写回 `localization.language`，复用已有 round-trip 测试保障重启持久化。
- Options Considered: 先不做语言交互；先做只读显示；先做 cycle 交互并持久化。
- Chosen Solution: 新增 `cycle_language_setting()` backend，并把 Slint callback 绑定到该持久化写回逻辑。
- Rationale: 这已经满足“可改、可存、界面立即刷新”的最小闭环，后续再升级成更明确的语言选择控件也不会推翻持久化设计。
