- Task: P1-04 将窗口尺寸固定为保守值，例如 `120 x 24`，确保肉眼容易识别
- Encountered Problem: 计划文档里的 `120 x 24` 更接近任务栏模块尺寸，但作为普通窗口基线过小，不利于早期肉眼确认。
- Thought Process: 保持“固定尺寸”这个约束，但把数值调到更适合 Phase 1 调试的范围。
- Options Considered: 严格使用 `120 x 24`；使用默认窗口尺寸；使用更保守的固定尺寸。
- Chosen Solution: 把普通窗口基线固定为 `240 x 80`。
- Rationale: 这仍然满足固定尺寸要求，同时降低 Phase 1 人工观察难度。
