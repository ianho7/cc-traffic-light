# P0-05 Reflection

- Task: 同步前端 AppConfig 类型。
- Encountered Problem: Tauri 设置页用 TypeScript 全量克隆和保存配置。
- Thought Process: Rust 与 TypeScript 字段必须同名并准确表达空绑定。
- Options Considered: `undefined`；宽松索引签名；显式 MaterialGroup 与 null。
- Chosen Solution: 新增 `MaterialGroup` interface 和两个 `string | null` 绑定。
- Rationale: 保持 IPC DTO 可检查，避免前端保存时遗失字段。
