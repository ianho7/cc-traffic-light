- Task: TWV1-B-01 Add an independent widget render module for transparent default lamps
- Encountered Problem: Phase 1 needed visibly cleaner circles and a transparent background without introducing a new heavyweight rendering stack.
- Thought Process: I chose a small software raster path that could render premultiplied ARGB buffers and still preserve the existing state semantics.
- Options Considered: Keep raw GDI ellipses; add a new external renderer; implement a small offscreen ARGB renderer in Rust.
- Chosen Solution: Implement `widget_render.rs` with supersampled circles, label glyphs, and layered-window frame submission.
- Rationale: It is light enough for the current scope and extensible enough to serve as the base for later visual work.
