- Task: P5-02 验证每轮都能重新找到句柄、完成嵌入、完成定位，没有偶发丢失
- Encountered Problem: 任务栏句柄、子窗口句柄和应用窗口句柄每次都会变化，必须验证流程本身稳定，而不是句柄值固定。
- Thought Process: 关注运行信号是否一致，而不是要求同一个十六进制句柄重复出现。
- Options Considered: 对比句柄值；对比 attach/move/destroy 信号；忽略重复性。
- Chosen Solution: 以 `Attached=true`、`Moved=true`、`Destroyed=true` 作为两轮一致性判断标准。
- Rationale: 对 Win32 窗口来说，这比静态句柄值更有意义。
