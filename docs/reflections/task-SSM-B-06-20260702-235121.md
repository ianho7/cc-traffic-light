- Task: SSM-B-06 建立第一版 Nothing-inspired 视觉 token
- Encountered Problem: 用户要求 settings 参考 Nothing 风格，如果只接框架不建立 token，容易退化成普通工具 UI。
- Thought Process: 参考 `$nothing-design` 的 tokens/components，把字体角色、单色基底、圆角和按钮形态先固化进静态骨架。
- Options Considered: 先用默认 Slint 样式；先落第一版 token 并保持范围只限 settings。
- Chosen Solution: 在 Slint 骨架中采用 Doto/Space Grotesk/Space Mono 角色分工、`#000/#111/#222/#333` 单色层级、16px 卡片圆角和无阴影结构化布局。
- Rationale: 这先把视觉方向钉住，后续做真实数据接线时不会因为默认样式而重新推翻结构。
