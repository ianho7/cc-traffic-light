- Task: SSM-F-03 确认 settings_window.rs 不再承担主入口职责
- Encountered Problem: 只保留 fallback 还不够，必须明确它已经不再是主入口，否则文档和实现会继续漂移。
- Thought Process: 从 `main.rs` 的 tray 打开路径和 host 初始化职责出发，确认默认展示权已经转移。
- Options Considered: 不写明角色变化；在 checklist 中显式记录“主入口已转移，Win32 仅 fallback”。
- Chosen Solution: 在 checklist 记录主入口转向 Slint runtime，Win32 window 仅承担 fallback 和短期兼容。
- Rationale: 这为后续清理和 handoff 提供清晰边界，避免团队把 Win32 设置页当成仍然是主线实现。
