- Task: CSW-C-08 confirm real state transitions cover `working` and `done`
- Encountered Problem: 仅有一条最终 `done` 记录不足以说明实时状态迁移链路在真实会话里工作。
- Thought Process: 需要同时看到当前主线程的 `working` 和已结束线程的 `done`，这样才能证明真实事件至少覆盖中间态和终态。
- Options Considered: 只接受 `done`；只接受 `working`；同时接受多个真实线程组成的状态样本。
- Chosen Solution: 使用当前主线程 `working` 和两个已结束线程 `done` 共同作为完成证据。
- Rationale: checklist 关注的是“至少覆盖 working，并在结束后看到 done 或允许的保守状态”，这组样本已经满足。
