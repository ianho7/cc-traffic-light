- Task: P5-03 检查退出时模块随进程消失，不残留悬挂窗口
- Encountered Problem: 强杀进程不能证明退出链路没问题，必须验证 `WM_CLOSE` 和消息循环退出。
- Thought Process: 在 `Shell_TrayWnd` 下找到应用子窗口，直接给它发 `WM_CLOSE`，再观察进程退出码和日志。
- Options Considered: 继续强杀；等待人工关闭；对子窗口发 `WM_CLOSE`。
- Chosen Solution: 通过子窗口句柄发送 `WM_CLOSE`，得到 `ExitCode=0`、`WM_DESTROY received`、`message loop exited cleanly`。
- Rationale: 这证明当前版本具备最小的优雅退出闭环。
