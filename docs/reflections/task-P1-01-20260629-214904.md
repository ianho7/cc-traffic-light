- Task: P1-01 新建独立 Rust bin 项目，并补齐最小 `windows` crate feature 集
- Encountered Problem: `cargo new` 只生成了空白 bin 项目，离 Win32 窗口基线还差实际依赖和模块骨架。
- Thought Process: 先用 `cargo new` 生成最小项目，再手动把依赖精确收敛到任务栏 MVP 所需的 Win32 feature。
- Options Considered: 手动从零写 Cargo 文件；使用 `cargo new` 后再精修；直接上更复杂 workspace。
- Chosen Solution: 使用 `cargo new taskbar-widget --bin`，然后在 `Cargo.toml` 中加入 `windows 0.58` 和最小 feature 集。
- Rationale: 这条路径最快，同时保持生成文件和依赖声明都可读。
