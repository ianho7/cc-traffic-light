# Task Reflection

- Task: `P2-02` 增加 `WS_EX_TOOLWINDOW`，但保持背景非透明
- Encountered Problem: 用户当前看到的是“日志成功但窗口不在任务栏里”，不是透明或渲染高级特性问题。
- Thought Process: 参考 TrafficMonitor 的初始化顺序，只拿对嵌入最有帮助、且不会引入透明变量的那一小步。
- Options Considered: 不加扩展样式；加 `WS_EX_TOOLWINDOW`；同时加分层透明样式。
- Chosen Solution: 只加 `WS_EX_TOOLWINDOW`，保持非透明。
- Rationale: 这保留了 MVP 的单变量原则，不把问题重新扩大到透明和分层。
