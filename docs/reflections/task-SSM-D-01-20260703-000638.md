- Task: SSM-D-01 定义 Slint 侧 view model，覆盖 Overview、General、Diagnostics 和语言选择字段
- Encountered Problem: 之前的 Slint 壳子只有静态文本属性，无法承接真实运行态和语言模式显示。
- Thought Process: 先围绕当前 MVP 真正需要展示的运行态信息扩展属性集，而不是一次把所有交互都塞进 view model。
- Options Considered: 保持最小静态属性；一次性设计完整交互模型；先补可展示的 live view model。
- Chosen Solution: 在 `settings.slint` 中增加 diagnostics 和 language 相关字段，让 Slint 先具备真实状态展示能力。
- Rationale: 这能让 Phase 4 形成真实的数据桥，而不是只有“能开窗口”的空壳。
