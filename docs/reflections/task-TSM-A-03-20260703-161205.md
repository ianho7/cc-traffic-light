- Task: TSM-A-03 列出 settings 页面真实字段和行为
- Encountered Problem: Tauri UI 很容易根据视觉稿多画出一层字段或交互，但当前正式数据面其实受 `AppConfig` 和 `AppStatusSnapshot` 限制。
- Thought Process: 我按 Overview/General/Monitoring/Appearance/Diagnostics/About 六页逐页核对当前真实字段、写入行为和即时生效要求。
- Options Considered: 只从 UI 看页面；只从 schema 看字段；把页面结构与 Rust 真字段对齐整理。
- Chosen Solution: 在基线文档中逐页列出真实字段和即时生效/轮询要求。
- Rationale: 这能防止 Tauri 直接消费内部结构或补出不存在的设置项。
