- Task: CSW-B-03 verify hook command paths and hook names
- Encountered Problem: `.codex/hooks.json` 一旦路径或 hook name 写错，真实 Codex 会话里通常表现为“完全不触发”，定位成本高。
- Thought Process: 用已有 `examples.codex-hooks.toml` 作为参照，再对修改后的 JSON 做一次 `ConvertFrom-Json` 结构检查。
- Options Considered: 只靠肉眼审查；跑 JSON 解析并对照示例；等 `/hooks` 失败后再排查。
- Chosen Solution: 同时做 JSON 解析检查和示例对照，确认路径、参数顺序、大小写都一致。
- Rationale: 这样能在真实 trust 之前先排除一类纯配置错误，降低下一阶段失败噪声。
