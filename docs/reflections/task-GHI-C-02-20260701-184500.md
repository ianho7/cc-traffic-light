- Task: GHI-C-02 implement apply with merge semantics
- Encountered Problem: 安装器不能整文件盲覆盖，否则会破坏用户已有 hooks。
- Thought Process: 把每个事件下的条目拆成“本软件受管”和“其他已有”两部分，再只替换受管部分。
- Options Considered: 整文件重写；只支持空配置；按事件 merge 并保留无关条目。
- Chosen Solution: `apply` 只写入 merge 后结果，并保留非本软件 hooks。
- Rationale: 这符合 P1 的核心安全目标，即 backup + merge 而不是覆盖。
