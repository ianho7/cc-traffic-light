- Task: GTV1-D-02 Implement fixed three-light mapping for each source
- Encountered Problem: The previous widget drew a single status blob with text, which did not satisfy the agreed traffic-light model of fixed green/yellow/red lamps per source.
- Thought Process: I introduced explicit per-source lamp rectangles and mapped each source state to one of the three lamp positions while keeping the overall taskbar footprint compact.
- Options Considered: Add text labels only; vary lamp count dynamically; keep a fixed three-light layout and choose which lamp is active.
- Chosen Solution: Keep a fixed three-light layout and choose which lamp is active.
- Rationale: That matches the agreed product model and makes later motion or color refinements possible without changing the structural layout again.
