# Task Reflection

- Task: `P1-01` 将正常运行路径的父窗口默认值固定为 `Shell_TrayWnd`
- Encountered Problem: 代码虽然默认已经偏向 `shell`，但语义上仍被“candidate parent”命名和矩阵诊断思路稀释。
- Thought Process: 先检查默认值来源，再把结构字段和运行语义改成明确的 `host_parent`。
- Options Considered: 保持 `candidate_parent` 只改文档；直接把运行时主字段改成 `host_parent`。
- Chosen Solution: 将主字段改为 `host_parent`，并让正常路径继续固定到 `Shell_TrayWnd`。
- Rationale: 这能让默认宿主路径在代码里直接可读，减少后续误判。
