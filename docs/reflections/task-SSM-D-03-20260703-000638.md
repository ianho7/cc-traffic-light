- Task: SSM-D-03 实现 AppConfig 到 Slint UI state 的状态推送
- Encountered Problem: Slint 如果只接 snapshot，不接 config，就无法展示当前开关值和语言模式。
- Thought Process: 让 config 与 snapshot 走同一条 bridge 更新路径，避免 UI 端再自行读取配置文件。
- Options Considered: 只在打开窗口时读取一次 config；把 config 也作为 live 更新输入。
- Chosen Solution: `settings_slint.rs` 在每次 show/update 时同时消费 `AppConfig`，填充 General 与 language 字段。
- Rationale: 这为后续 Phase 5 的交互回写提供了对称结构，也保证窗口重开时不会显示旧值。
