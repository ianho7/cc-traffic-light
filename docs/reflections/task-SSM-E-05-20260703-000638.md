- Task: SSM-E-05 把 tray tooltip 改为通过 i18n 层输出
- Encountered Problem: tooltip 之前用英文拼接内部状态 key，既不国际化也不适合作为最终用户文案。
- Thought Process: 利用共享 i18n 层直接输出聚合摘要，同时保留 `overall/codex/claude` 这套信息结构。
- Options Considered: 只翻译标题；把整条 tooltip 都切到共享本地化摘要生成函数。
- Chosen Solution: 在 `i18n.rs` 中增加 `tray_tooltip()`，由 `tray_icon.rs` 统一调用。
- Rationale: 这满足了“通过 i18n 层输出且摘要语义不变”的要求，也让 tooltip 与 settings 共享同一套状态标签。
