# Task Reflection

- Task: `P2-01` 将顶层窗口风格改成更适合嵌入的保守样式
- Encountered Problem: 旧窗口以 `WS_OVERLAPPED + WS_CAPTION + WS_MINIMIZEBOX` 创建，天然更像普通桌面浮窗。
- Thought Process: 当前目标不是最终视觉，而是尽量减少嵌入前的顶层窗口语义。
- Options Considered: 保留原风格只在 `SetParent` 后覆盖；创建时就改成 `WS_POPUP | WS_VISIBLE`。
- Chosen Solution: 创建时直接使用 `WS_POPUP | WS_VISIBLE`。
- Rationale: 这最直接地削弱普通浮窗特征，有利于后续人工判断。
