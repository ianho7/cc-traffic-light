- Task: SSM-C-05 为配置 round-trip 和语言资源完整性补测试

- Thought Process: 优先补最小但高价值的单测，覆盖默认回退、显式 round-trip、localizer 行为和资源 key 完整性。
- Options Considered: 只依赖人工检查；增加四个低成本单测先卡住基础契约。
- Chosen Solution: 增加 4 个库测试，并用 `cargo test --lib` 验证通过。
- Rationale: 这些测试足以为后续继续接运行时 UI 和 tray 行为提供基础回归保护。
