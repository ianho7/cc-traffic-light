- Task: CCH-A-02 设计最小 shape-only Claude hooks 配置
- Encountered Problem: 需要覆盖首批重点事件，同时保证所有 handler 都只走 shape-only 采样路径。
- Thought Process: 选择 `UserPromptSubmit`、`PreToolUse`、`PermissionRequest`、`Notification`、`Stop`、`StopFailure` 六个事件，并统一把 command 指向 `taskbar_widget_hook.exe sample`。
- Options Considered: 只配置 1 到 2 个事件；首批事件一次性全配齐。
- Chosen Solution: 首批事件一次性全配齐，但都指向同一个 sample 入口。
- Rationale: 这样可以减少后续频繁改配置，同时保持采样路径一致。
