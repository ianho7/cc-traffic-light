- Task: TLU-C-04 用 debug set 验证 `done` 的颜色、形态和可读性
- Encountered Problem: `done` 优先级较低，如果状态目录不干净容易被更高优先级状态覆盖。
- Thought Process: 清理状态后只保留一个 `done` 任务，确认绿色完成态能够直接显示。
- Options Considered: 从混合状态中推断 `done`；隔离场景下直接测。
- Chosen Solution: 隔离场景下执行 `set codex_123 done`。
- Rationale: 这能确认完成态在当前 UI 中既可见又没有被其他状态逻辑吞掉。
