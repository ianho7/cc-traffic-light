- Task: SSM-C-02 将语言设置纳入持久化并定义默认/跟随系统策略
- Encountered Problem: 语言如果不进入配置模型，只能做到临时切换，无法满足重启后保持一致。
- Thought Process: 在实现简单和扩展性之间取平衡，优先采用三态语言模式并定义明确 fallback。
- Options Considered: 只存显式 locale；只跟随系统；三态 `follow_system` / `zh-CN` / `en`。
- Chosen Solution: 使用三态语言枚举，默认 `follow_system`，旧配置缺失时也回落到该值。
- Rationale: 这同时满足默认跟随系统和用户手动覆盖两种路径，而且序列化最直接。
