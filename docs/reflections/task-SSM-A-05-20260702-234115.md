- Task: SSM-A-05 确定语言策略字段进入配置模型
- Encountered Problem: 没有持久化的语言策略字段，就无法满足重启后保持语言一致的要求。
- Thought Process: 在满足用户需求和实现简单之间取平衡，优先选择可枚举、易序列化的语言模式。
- Options Considered: 仅存具体 locale；只跟随系统不允许手动覆盖；采用 `follow_system`/`zh-CN`/`en` 的枚举型表示。
- Chosen Solution: 建议把语言策略建模为枚举型配置值，至少支持 `follow_system`、`zh-CN`、`en`，解析失败回退 `follow_system`。
- Rationale: 这同时覆盖默认跟随系统与用户手动覆盖两种需求，且最适合后续统一 i18n 层消费。
