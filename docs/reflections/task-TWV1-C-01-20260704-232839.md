- Task: TWV1-C-01 Define per-group rectangular hot zones
- Encountered Problem: A transparent widget cannot keep using the whole window as an implicit hit target without interfering with the taskbar.
- Thought Process: I made the render layer produce explicit hot-zone rectangles in the same coordinate space as the visual layout.
- Options Considered: Keep whole-window hit testing; rely on pixel alpha only; store per-group rectangles and use `WM_NCHITTEST`.
- Chosen Solution: Store per-group hot zones and return `HTTRANSPARENT` outside them.
- Rationale: This keeps single-group and dual-group interaction aligned with layout while avoiding invisible full-window click interception.
