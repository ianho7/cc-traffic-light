- Task: CCH-PRE-07 确认优先使用项目本地实验配置
- Encountered Problem: P2 需要真实 Claude hooks，但当前仓库里没有 `.claude/`，而全局配置又不该作为默认第一步。
- Thought Process: 先检查仓库内 `.claude/` 是否存在；发现不存在后，将实验入口收敛到项目本地 `.claude/settings.local.json`。
- Options Considered: 直接改全局用户配置；先在项目内创建本地实验配置。
- Chosen Solution: 使用项目本地实验配置。
- Rationale: 这样范围更小，风险更低，也符合集成说明里“个人调试优先 settings.local”。
