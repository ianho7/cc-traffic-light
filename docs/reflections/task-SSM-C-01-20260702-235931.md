- Task: SSM-C-01 扩展 AppConfig 以覆盖正式 settings schema
- Encountered Problem: 当前计划中的语言设置还没有持久化落点，无法满足“重启后保持一致”。
- Thought Process: 在保持旧配置兼容的前提下，为 `AppConfig` 增加独立的本地化配置块，并确保缺失字段有默认值。
- Options Considered: 把语言字段塞进已有 general/appearance；新增 `localization` 配置块。
- Chosen Solution: 新增 `localization.language`，并通过 serde default 保证旧配置可平滑加载。
- Rationale: 独立配置块语义最清晰，也为后续继续增加语言相关策略预留了位置。
