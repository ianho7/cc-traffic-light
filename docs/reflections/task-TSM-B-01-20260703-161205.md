- Task: TSM-B-01 建立根级 Cargo workspace
- Encountered Problem: 后续 `shared-core` 和 `src-tauri` Rust crate 都需要 workspace，但当前仓库还只有 `taskbar-widget` 单 crate。
- Thought Process: 我先做最小 workspace，只把现有 crate 纳入，避免在还没创建 shared-core 时就引入多成员构建复杂度。
- Options Considered: 暂不建 workspace；一次性把未来 crate 名称都写进去；先建立只含 `taskbar-widget` 的 workspace。
- Chosen Solution: 新增根级 `Cargo.toml`，声明 `[workspace] members = [\"taskbar-widget\"]`。
- Rationale: 这给后续迁移预留结构位置，同时保持当前构建路径基本不变。
