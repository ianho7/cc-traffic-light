- Task: GHI-D-04 run restore against fixtures
- Encountered Problem: backup 存在不代表 restore 真能回到原状态。
- Thought Process: restore 需要覆盖已有文件和原本不存在文件两类场景。
- Options Considered: 只看 backup 文件是否生成；只测已有文件；把 restore 也放进 fixture 闭环。
- Chosen Solution: 对 fixture 和 missing-file 场景都实际执行 `restore`。
- Rationale: 这验证了 backup meta 的存在意义，也把回滚语义跑成了闭环。
