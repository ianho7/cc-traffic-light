- Task: PRE-01 固定目标环境，记录当前主力机器的 Win11 版本、任务栏对齐方式、缩放比例
- Encountered Problem: `Get-ComputerInfo` 返回 `Windows 10 Pro / 2009`，与用户口述的 Win11 不一致，单看这个字段会误导后续路径选择。
- Thought Process: 先把系统品牌名和真实 build 拆开验证，再补查 `DisplayVersion`、`CurrentBuild` 和系统 DPI，避免拿到伪线索就开始写 Win11 专属逻辑。
- Options Considered: 直接相信用户口述；直接相信 `Get-ComputerInfo`；以注册表 build 和 DPI 结果作为主证据。
- Chosen Solution: 采用 `DisplayVersion 25H2`、`CurrentBuild 26200`、`GetDpiForSystem() = 96` 作为环境基线，并把 `TaskbarAl` 缺失记录为待手动确认项。
- Rationale: 任务栏嵌入依赖实际系统结构，环境证据必须优先于口述或容易误导的品牌字段。
